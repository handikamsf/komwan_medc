# Cloud-Based Risk Management: Forecasting Saham Energi (MEDC) Merespons Anomali Geopolitik via Pipeline AWS

Proyek ini adalah implementasi *End-to-End Data Pipeline* di lingkungan komputasi awan (AWS) untuk memprediksi harga saham PT Medco Energi Internasional Tbk (MEDC) dan mendeteksi anomali volatilitas yang disebabkan oleh guncangan makroekonomi/geopolitik.

Proyek ini merupakan Evaluasi Tengah Semester (ETS) untuk mata kuliah Komputasi Awan, Program Studi Sains Data, UPN "Veteran" Jawa Timur.

## ⚙️ Arsitektur & Teknologi yang Digunakan
* **Data Lake:** Amazon Simple Storage Service (S3) (Konsep *Decoupled Architecture*)
* **Compute Instance:** Amazon SageMaker Notebook
* **Programming Language:** Python 3 (Pandas, Boto3, Matplotlib)
* **Analitik:** Interquartile Range (IQR) untuk Deteksi Anomali
* **Machine Learning:** Long Short-Term Memory (LSTM) untuk Forecasting
